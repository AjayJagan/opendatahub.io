What is Open Data Hub?

The Open Data Hub is a collection of open source and cloud components deployed as a "machine-learning-as-a-service" platform. Users can build, deploy and execute analytic, machine learning and AI models. Here, the Open Source Community can experiment and develop Intelligent Applications without incurring high costs and without having to master the complexity of modern AI Software Stacks.

Open Data Hub is a Meta-Project to integrate Open Source projects into a practical solution. It aims to foster collaboration between communities, vendors, user-enterprises, and academics following Open Source best practices. By focusing on integration, service abstraction, and continuity, OpenDataHub frees users from owning the full complexity of the underlying stack. In addition, it ensures reproducibility by using hard Open Source.

There are no proprietary products and no lock-in. We offer flexible entry points and enable related projects to pick and choose services.
